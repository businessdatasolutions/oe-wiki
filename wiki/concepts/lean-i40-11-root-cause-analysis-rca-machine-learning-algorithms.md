# Root Cause Analysis (RCA) ↔ Machine Learning Algorithms

Referentiepagina 11/23 uit Gomaa (2025) Tabel VII: de 23 gedocumenteerde koppelingen tussen een
klassiek Lean-tool en een Industrie-4.0-technologie.

## De koppeling (Tabel VII)

| Lean-tool | Industrie-4.0-technologie | Objective |
|---|---|---|
| Root Cause Analysis (RCA) | Machine Learning Algorithms | Use advanced analytics to identify root causes and predict potential failures. |

**Root Cause Analysis (RCA)** (Tabel I): Identify and resolve root causes of problems.

## Hoe de koppeling werkt

RCA identificeert en verhelpt onderliggende oorzaken van operationele problemen. Machine-learning-algoritmes analyseren historische data en patronen om verborgen oorzaken bloot te leggen en storingen te voorspellen, wat proactieve besluitvorming mogelijk maakt.

## Chapter-tag en redenering

- **Hoofdstuk:** Ch15
- **D-fase:** develop
- **Redenering:** RCA is het expliciete verbetermechanisme in de 'Kaizen at Amazon'-case (Ch15): de scannerbatterij-storing werd via root-cause-analyse opgelost, niet via een generieke Lean-tool uit Ch16 (zie [[amazon]]).
